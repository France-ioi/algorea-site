Comment participer au concours Algoréa ?

La participation en classe, organisée par un enseignant coordinateur, est à privilégier. Des modalités de participation à la maison existent aussi.

Accéder aux parcours du concours

Si vous avez déjà votre code de participant ou bien un code de groupe fourni par un enseignant, allez directement sur la plateforme du concours et entrez votre code pour démarrer.

S'inscrire à Algoréa

Trois manières de rejoindre Algoréa !

Participer via un code de groupe

Cette modalité d'inscription, identique à celle du concours Castor, est réservée aux enseignants qui organisent le concours en classe.

La participation via un code de groupe n'est possible que pour les catégories blanche et jaune.

Participer via un code individuel de participant

Avoir participé au concours Castor donne l'accès à Algoréa en catégorie blanche ou jaune selon le score obtenu.

L'accès aux parcours se fait via un code de participant Algoréa individuel, que les enseignants qui ont inscrit leurs élèves au concours Castor peuvent générer à partir de leur interface coordinateur dès la fin de celui-ci.

Le code de participant individuel peut être utilisé en classe ou à la maison, suivant le paramétrage effectué dans l'interface coordinateur.

Participer via un compte sur france-ioi.org

Avoir un compte sur france-ioi.org et résoudre au minimum 12 exercices du parcours général permet de s'inscrire à Algoréa.
La catégorie accessible dépend du nombre d'exercices validés :

Blanche Avoir validé 12 exercices
Jaune Avoir validé 20 exercices
Orange Avoir validé 50 exercices
Verte Avoir validé le niveau 2 du parcours général

Allez ensuite sur cette page pour obtenir votre code de participant Algoréa.

Utiliser un code de participant Algoréa

  • Un code individuel de participant Algoréa est constitué d'une douzaine de caractères.
  • Pour accéder à un parcours, il faut l'entrer sur la page d'accueil de la plateforme du concours.
  • Le code de participant est à conserver pendant toute la durée du concours. Il faut le garder secret. Il permet au participant d'accéder au concours dans la bonne catégorie, de visualiser ses participations et résultats.
  • Une même personne ne doit jamais utiliser plus d'un code de participant, sous peine de disqualification. Une vérification systématique est effectuée pour l'accès à la phase finale.

Enseignants

L'organisation de la participation de vos élèves au concours Algoréa se fait en utilisant la même interface coordinateur que pour le concours Castor. Vous y trouverez un document pour vous accompagner pas à pas et gérer les différents aspects comme la participation à la maison, les absents le jour d'une épreuve en classe...

Les résultats de vos élèves, et leur catégorie Algoréa, sont consultables dans cette interface coordinateur après la publication des résultats du Castor, puis après chaque tour d'Algoréa.

Notez que vous pouvez vous inscrire comme coordinateur à tout moment, y compris si vous n'avez pas inscrit vos classes au concours Castor.

 L'organisation du concours Algoréa est flexible, en fonction du temps de classe que vous souhaitez y consacrer.
